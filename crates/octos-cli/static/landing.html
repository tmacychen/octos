Up and running in 4 steps

四步开始使用

Register an account, name your node, bootstrap your Mac Mini, and your private AI dashboard is live.

注册账号、命名节点、配置 Mac Mini，你的私有 AI 面板即刻上线。

Register Account

注册账号

Sign up with your email. We send a one-time code to verify—no password needed.

使用邮箱注册，我们发送一次性验证码确认身份，无需设置密码。

Name Your Node

命名节点

Choose a subdomain for your Octos node (e.g. alice.octos-cloud.org). We generate your auth token, SSH tunnel port, and public address for this registered machine, and your operator provides the shared FRPS token.

为你的 Octos 节点选择子域名（如 alice.octos-cloud.org），系统会为这台已注册机器生成认证令牌、SSH 隧道端口和公网地址，共享 FRPS 令牌则由运维方提供。

Setup Your Machine

配置你的机器

Run the one-line setup command on your Mac, Linux server, or Windows PC. It installs octos, configures the tunnel, and brings your public node online.

在你的 Mac、Linux 服务器或 Windows 电脑上运行一行安装命令，自动安装 octos、配置隧道，并让你的公网节点上线。

Go Live

上线运行

Your dashboard is live at your subdomain. Add API keys, connect channels, start chatting with your AI agent.

面板即刻上线。添加 API 密钥、接入消息渠道，开始与你的 AI 智能体对话。

Architecture架构

How it connects

连接方式

Your Mac Mini connects outbound to our relay. No port forwarding or public IP needed.

你的 Mac Mini 主动向中继服务器建立出站连接，无需端口转发或公网 IP。 

  Internet
     |
     v
  Caddy  (octos-cloud.org, SSL)
     |
     v
  frps   (tunnel relay隧道中继)
     |         |         |
     v         v         v
  alice     bob       carol

⚡

Rust-Native Performance

Rust 原生性能

Single ~30MB binary. Starts in milliseconds. Hundreds of concurrent sessions, minimal memory.

单个 ~30MB 二进制文件，毫秒级启动，支持数百并发会话，内存占用极低。

🔄

Auto-Failover

自动故障切换

Primary + fallback providers. 401/403 failover, 429/5xx retry then failover. Always online.

主备模型供应商配置。401/403 自动切换，429/5xx 先重试再切换。确保永远在线。

📋

Multi-Profile

多配置文件

Multiple agent profiles per machine, each with own provider, channels, and data. Fully isolated.

一台设备运行多个智能体配置，各自独立的模型、渠道和数据目录，完全隔离。
